MySQL Commands

# Descreption Command
1 To login (from unix shell) use -h only if needed. [mysql dir]/bin/mysql -h hostname -u root -p
2 Create a database on the sql server. create database [databasename];
3 List all databases on the sql server. show databases;
4 Switch to a database. use [db name];
5 To see all the tables in the db. show tables;
6 To see database's field formats. describe [table name];
7 To delete a db. drop database [database name];
8 To delete a table. drop table [table name];
9 Show all data in a table. SELECT * FROM [table name];
10 Returns the columns and column information pertaining to the designated table. show columns from [table name];
11 Show certain selected rows with the value "whatever". SELECT * FROM [table name] WHERE [field name] = "whatever";
12 Show all records containing the name "Bob" AND the phone number '3444444' SELECT * FROM [table name] WHERE name = "Bob" AND phone_number = '3444444';
13 Show all records not containing the name "Bob" AND the phone number '3444444' order by the phone_number field. SELECT * FROM [table name] WHERE name != "Bob" AND phone_number = '3444444' order by phone_number;
14 Show all records starting with the letters 'bob' AND the phone number '3444444'. SELECT * FROM [table name] WHERE name like "Bob%" AND phone_number = '3444444';
15 Use a regular expression to find records. Use "REGEXP BINARY" to force case-sensitivity. This finds any record beginning with a. SELECT * FROM [table name] WHERE rec RLIKE "^a$";
16 Show unique records. SELECT DISTINCT [column name] FROM [table name];
17 Show selected records sorted in an ascending (asc) or descending (desc). SELECT [col1],[col2] FROM [table name] ORDER BY [col2] DESC;
18 Count rows. SELECT COUNT(*) FROM [table name];
19 Join tables on common columns. select lookup.illustrationid, lookup.personid,person.birthday from lookup left join person on lookup.personid=person.personid=statement to join birthday in person table with primary illustration id;
20 Switch to the mysql db. Create a new user. INSERT INTO [table name] (Host,User,Password) VALUES('%','user',PASSWORD('password'));
21 Change a users password.(from unix shell). [mysql dir]/bin/mysqladmin -u root -h hostname.blah.org -p password 'new-password'
22 Change a users password.(from MySQL prompt). SET PASSWORD FOR 'user'@'hostname' = PASSWORD('passwordhere');
23 Switch to mysql db.Give user privilages for a db. INSERT INTO [table name] (Host,Db,User,Select_priv,Insert_priv,Update_priv,Delete_priv,Create_priv,Drop_priv) VALUES ('%','db','user','Y','Y','Y','Y','Y','N');
24 To update info already in a table. UPDATE [table name] SET Select_priv = 'Y',Insert_priv = 'Y',Update_priv = 'Y' where [field name] = 'user';
25 Delete a row(s) from a table. DELETE from [table name] where [field name] = 'whatever';
26 Update database permissions/privilages. FLUSH PRIVILEGES;
27 Delete a column. alter table [table name] drop column [column name];
28 Add a new column to db. alter table [table name] add column [new column name] varchar (20);
29 Change column name. alter table [table name] change [old column name] [new column name] varchar (50);
30 Make a unique column so you get no dupes. alter table [table name] add unique ([column name]);
31 Make a column bigger. alter table [table name] modify [column name] VARCHAR(3);
32 Delete unique from table. alter table [table name] drop index [colmn name];
33 Load a CSV file into a table. LOAD DATA INFILE '/tmp/filename.csv' replace INTO TABLE [table name] FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n' (field1,field2,field3);
34 Dump all databases for backup. Backup file is sql commands to recreate all db's. [mysql dir]/bin/mysqldump -u root -ppassword --opt >/tmp/alldatabases.sql
35 Dump one database for backup. [mysql dir]/bin/mysqldump -u username -ppassword --databases databasename >/tmp/databasename.sql
36 Dump a table from a database. [mysql dir]/bin/mysqldump -c -u username -ppassword databasename tablename > /tmp/databasename.tablename.sql
37 Restore database (or database table) from backup. [mysql dir]/bin/mysql -u username -ppassword databasename < /tmp/databasename.sql
